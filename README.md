# Upi
🛡️ UPI Fraud Detection System

A modern, responsive web application that helps users analyze UPI transactions and estimate the risk of fraud before making a payment. The system validates UPI IDs, detects the bank/provider, evaluates transaction amount risk, and generates a fraud risk score with recommendations.

✨ Features
✅ Modern & Responsive User Interface
✅ UPI ID Format Validation
✅ Bank/Provider Detection
✅ Transaction Amount Risk Analysis
✅ Fraud Risk Score (0–100%)
✅ Final Transaction Result

⚙️ How It Works
Enter the Receiver's UPI ID.
Enter the transaction amount.
Click Analyze Transaction.
The system validates the UPI ID.
Detects the bank/provider.
Calculates the fraud risk score.
Displays:
          UPI Status
           Bank Name
           Amount Risk
           Fraud Risk Score
           Final Result
           Recommendation
Saves the latest 3 analyses in Local Storage
